# Trilha_de_Python_2026.1
 O código foi construído no programa vs code, com a finalidade de apresentar um programa que calcule as finanças de viagem de uma cliente hipotética, 
 utilizando o git hub como ferramenta de visualização e entrega do código.
 #Comentários sobre a elaboração do código.
 O sistema permite a conversão de moedas (euro para real).
 Cáculo de gastos previstos, como hospedagem e passagem.
 Seguindo a restrição alocada na elaboração do desafio, foi utilizado o tratamento de erro, que impede que o programa prossiga, caso possua algum número negativo nele, garantindo a integridade dos cálculos.
 print: ("ValueError")
 Os operadores lógicos, como and, if e else foram utilizados para elucidar as saídas do código no programa e auxiliar 
 que as tomadas de decisões complexas sejam executadas plenamente.

Respostas as questões teóricas:
1: Qual a diferença entre o comando git add e git commit - m "mensagem"?
O git add: adiciona todas as preparações do diretório atual. Já o git commit - m é uma estrutura de dados imutáveis que agrupa todas as alterações para enviar ao repositório.
2: Por que é necessário realizar o casting (conversão de tipo) ao usar a função input( ) em python para cálculos matemáticos?
Porque para realizar os cálculos matemáticos devemos usar números inteiros ou float. 
Com a função input, o Python entende que se trata de resolução de números e executa o comand. "O input salva como número".
3: O que acontece se tentarmos somar uma variável do tipo str com uma do tipo float?
Ocorre um erro, pois não se pode realizar uma adição com um número e uma letra.
